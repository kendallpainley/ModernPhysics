# Modern Physics

A community study resource breaking down key chapters regarding Modern Physics.
Each chapter covers the core concepts, walks through the worked
examples, and tackles selected problems.

##  Format

Each chapter lives in its own folder and follows the same layout:
```
/Chapter 5/
│
├── relativity_overview.ipynb        # Main notebook
│
├── conceptual_questions.ipynb
│
├── examples/
│   ├── example_5_x.ipynb
│   └── ...
│
└── problems/
    ├── problem_5_x.ipynb
    └── ...
```
